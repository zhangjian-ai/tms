# 云设备管理平台 - 前端

## 项目简介

这是云设备管理平台的前端项目，基于 Vue 3 + Element Plus 开发，提供设备管理的完整功能。

## 功能特性

- 🖥️ **设备列表管理** - 支持分页、搜索、筛选
- ➕ **新增设备** - 完整的设备信息录入表单
- 👁️ **设备详情查看** - 详细的设备信息展示
- 🔍 **多条件搜索** - 支持按名称、序列号、品牌、操作系统等搜索
- 📱 **响应式设计** - 适配不同屏幕尺寸
- 🎨 **现代化UI** - 基于Element Plus的美观界面

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vue Router 4** - Vue 3 官方路由管理器
- **Element Plus** - Vue 3 版本的 Element UI 组件库
- **Axios** - HTTP 客户端
- **Vue CLI** - Vue 项目脚手架

## 安装和运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run serve
```

项目将在 `http://localhost:8080` 启动

### 3. 构建生产版本

```bash
npm run build
```

### 4. 代码检查

```bash
npm run lint
```

## API 配置

前端默认连接后端服务地址：`http://localhost:8888`

如需修改，请编辑 `src/config/index.js` 文件中的配置项。

### 环境配置

项目支持多环境配置：
- **开发环境**：`development`
- **生产环境**：`production`
- **测试环境**：`test`

可以通过修改 `src/config/index.js` 文件来配置不同环境的API地址。

## 功能说明

### 主页面布局

- **侧边栏导航**：可折叠的侧边栏，专注于设备管理功能
- **顶部导航栏**：面包屑导航和用户信息
- **响应式设计**：适配不同屏幕尺寸
- **路由管理**：基于 Vue Router 的单页面应用

### 设备管理页面

- **搜索功能**：支持按设备名称、序列号、品牌、操作系统、系统版本、状态进行搜索
- **分页功能**：支持自定义每页显示数量，支持页码跳转
- **表格展示**：展示设备的完整信息，包括状态标签
- **操作功能**：支持查看设备详情

### 新增设备

- **表单验证**：必填字段验证
- **枚举选择**：操作系统和状态使用下拉选择
- **数字输入**：屏幕尺寸使用数字输入框
- **提交反馈**：成功/失败消息提示

### 设备详情

- **信息展示**：使用描述列表展示设备详细信息
- **状态标签**：操作系统和状态使用彩色标签显示
- **时间格式**：创建时间和更新时间格式化显示

## 开发说明

### 组件开发

- 使用 Vue 3 Composition API

- 响应式数据管理
- 异步操作处理

### 样式设计

- 使用 Element Plus 默认主题
- 响应式布局

## 注意事项

1. 确保后端服务已启动并运行在 `http://localhost:8888`
2. 如果后端服务地址变更，请修改 `src/api/device.js` 中的 `baseURL`
3. 确保网络连接正常，能够访问后端 API
4. 建议使用现代浏览器以获得最佳体验
